### 3. NSD (Network Shared Disk)
#### 3-1) Create NSD 
```
nano nsd      
    %nsd:
     device=/dev/sdb
     nsd=nsd1
     servers=node01
     usage=dataAndMetadata
     failuserGroup=1
     pool=system

    %nsd:
     device=/dev/sdc
     nsd=nsd2
     servers=node01
     usage=dataOnly
     failuserGroup=1
     pool=data1
```

```
mmcrnsd -F ./nsd       //nsd 생성
mmmount -a       // 전체 mount
mmlsnsd        // nsd 리스트
```
